System- und Schnittstellenkonfiguration
Lexware Office – API-Anbindung
Rechnungen direkt aus Reparaturaufträgen in Lexware erstellen
Token-Verwaltung: app.lexware.de › Addons › Public API ↗
Wird als Voreinstellung beim Rechnungserstellen verwendet
So funktioniert die Integration
1. API-Token aus Lexware → Addons → Public API kopieren und hier einfügen
2. In einem Reparaturauftrag auf „Rechnung erstellen" klicken
3. Positionen + Preise prüfen und direkt an Lexware übertragen
4. Kunde wird automatisch in Lexware angelegt, Rechnung sofort verfügbar
helloCash – Kassensystem API
Rechnungen direkt aus Reparaturaufträgen in helloCash erstellen
Token-Verwaltung: myhellocash.com › Kasse › Einstellungen ↗
Wird als Voreinstellung beim Rechnungserstellen verwendet
So funktioniert die Integration
1. API-Token aus helloCash → Kasse → Einstellungen → API generieren
2. In einem Reparaturauftrag auf „helloCash" klicken
3. Positionen + Preise prüfen und direkt an helloCash übertragen
4. Rechnung wird sofort in helloCash erstellt und kann ausgedruckt werden
IMEI.info – Geräte-Datenbank
IMEI-Nummern automatisch abfragen: Marke, Modell, Gerätedaten
Account erstellen: imei.info › API Dokumentation ↗
